## 4. 핵심 기능
### 수업일정표 · 강의안
- 15회차 일정(`snuSchedule.ts` + DB `snu_schedule`): 회차·일자·강사·교육방식(온라인 7/오프라인 고정 3).
- **강의안(Day01~15)**: 학습목표(성취기준)·학습내용·활동·과제 + 강의 세션 11회차의 **전문 강의 노트**
  (조작적 정의·EDA·삼각측정·트랜스포머/RAG·툴민 논증·인과추론 등 학술/기술 서술). 강사 PT 제작용.

### 개인별 PBL 활동 (자동 평가)
- 기본정보(이름·학번·연락처) + **소속(대학·학과·전공)·트랙 4가지로 수강 명단 대조**, 회원정보 반영.
- 6단계 워크시트(문제정의→팀빌딩→중간설계→멘토링→구현→고도화), 작성 즉시 **휴리스틱 자동 채점**
  (`promptEval.ts`: 분량·구체성·구조화·핵심요소→0~100), 내용+점수 DB 저장(`snu_pbl_submissions`).
- 단계별 점수·총점(100점)·격려 메시지, 강사 평가(수동) 병행, 평가 루브릭(CT 7단계).

### 자가평가 (성적 미반영)
- 사전·진단·사후 3종, 각 20문항 객관식. 번호 네비게이터·즉시 채점.
- 사전/사후: 채점·저장·응시 3회 / 진단: 자습(정답·해설 공개, 무제한). `snu_assessments` 재사용.

### 프로젝트
- 아이디어 예시(서울 7·제주 7, 난이도·트랙·데이터·산출물·확장), 아이디어 구현(화면설계·DB 스키마·구현 가이드).
- 팀구성: 학생 자체 주제 등록·투표·팀장 선정(트랙별 정원), 프로젝트 게시판.

### LMS · 운영
- 공지·자료·과제·Q&A·팀게시판, 관리자(명단·평가) 화면. 공지 3건 시드.
